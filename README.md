# seguridad_lab
Este repositorio documenta mi proceso de formación en ciberseguridad,
 abarcando desde fundamentos de Linux y redes hasta técnicas avanzadas
 de Pentesting y Blue Teaming.
 El objetivo es consolidar conocimientos técnicos y construir una base sólida para
 certificaciones como eJPT y OSCP.
